# Road Accident Data Analysis

##  Project Overview
This project analyzes road accident data to identify patterns, trends, and key factors affecting accidents, injuries, and fatalities across different states and years in India.
The analysis is performed using **Python, Excel, and Power BI** to gain meaningful insights and build an interactive dashboard.

---

##  Objectives
- Analyze state-wise accident distribution  
- Study year-wise trends in accidents, injuries, and deaths  
- Identify high-risk states  
- Understand the relationship between accidents and fatalities  
- Calculate and analyze death rate  

---

## Tools & Technologies Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- Excel  
- Power BI  

---

##  Dataset Source
The dataset is obtained from official government sources:

- Ministry of Road Transport and Highways (MoRTH)  
- Data.gov.in  

It includes state-wise and year-wise data on road accidents, injuries, and fatalities.

---

## Project Workflow

### 1. Data Collection
- Dataset downloaded from Data.gov.in  

### 2. Data Cleaning
- Handled missing values  
- Converted data types  
- Structured dataset for analysis  

### 3. Feature Engineering
- Created:
  - Total Accidents  
  - Total Deaths  
  - Total Injured  

### 4. Exploratory Data Analysis (EDA)
- State-wise analysis  
- Year-wise trends  
- Correlation analysis  

### 5. Visualization
- Bar charts  
- Line graphs  
- Scatter plots  

### 6. Dashboard Creation
- Built interactive dashboard in Power BI  

---

##  Key Insights

- States like Tamil Nadu, Madhya Pradesh, and Uttar Pradesh report the highest number of accidents.  
- Uttar Pradesh has the highest number of fatalities, indicating higher accident severity.  
- A strong positive relationship exists between accidents and deaths.  
- Accidents decreased in 2020 and increased again in 2021, showing the impact of external factors.  
- Death rate varies across states, highlighting differences in safety conditions.

---

## Power BI Dashboard Features
- KPI Cards (Total Accidents, Deaths, Injured, Death Rate)  
- Year-wise trend analysis  
- Top 10 states comparison  
- Accidents vs Deaths relationship  
- Interactive filters (Year, State)
